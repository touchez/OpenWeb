# HOW TO USE

1. put `testServer.py` and `OpenWeb.py` in same dir. 

2. change the server ip and port below

   ```python
   # testClient.py
   SERVER_IP = "localhost"
   SERVER_PORT = 16000
   
   # testServer.py
   HOST_IP = ""
   HOST_PORT = 16000
   ```

3. run `testServer.py ` and `testClient.py`
